# 🚀 YouTube Video Analyzer & Downloader (Python)

✨ Analyze any YouTube video and download it in the best quality — all from a simple Python script with a smooth CLI + folder picker UI.

---

## 🔥 Features

- 📊 Fetch detailed video statistics
- 🎯 Clean formatted output (commas, duration, date)
- 🎥 Download videos in best available quality
- 📂 Select download folder using GUI (Tkinter)
- ⚡ Fast & lightweight
- 🛡 Error handling included

---

## 📊 Video Insights You Get

- 🎬 Title  
- 📺 Channel Name  
- 👀 Views (formatted with commas)  
- 👍 Likes  
- 💬 Comments  
- ⏱ Duration (mm:ss)  
- 📅 Upload Date (YYYY.MM.DD)  

---

## 🛠 Tech Stack

- Python 🐍  
- yt-dlp  
- tkinter  

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/garvit1226/YouTube-Video-Analyzer-Downloader.git
cd YouTube-Video-Analyzer-Downloader
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the script:

```bash
python main.py
```

### 🔄 Workflow

1. Enter YouTube URL  
2. View video stats  
3. Choose to download (yes/no)  
4. Select folder  
5. Video gets downloaded
